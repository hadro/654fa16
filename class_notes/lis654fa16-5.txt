JPG - Really good compression.
PNG - Portable network graphics
GIF - Older but good compression for up to 256 colorspace.
BMP - Don't talk about BMP
TIFF - Tag Image File Format. Extremely stable and well documented image format. Technically owned by Adobe
JPEG2000 - useful compression format
