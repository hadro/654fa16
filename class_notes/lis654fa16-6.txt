    Between XML and HTML
    
    The same:
        Paired element tags
        Doctype and dtd
        Same orthography
        Elements and attributes
        Always a root element
        Nested elements
        Attributes must be quoted
        
    Different:
	    Roll your own elements
		Roll your own attributes
		Function less deterministic
		White space is not truncated
		Element names are case sensitive
